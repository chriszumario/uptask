# Administrador de Tareas y Proyectos (MERN Stack) 🚀

Este proyecto es un **Administrador de Tareas y Proyectos** construido con el stack **MERN** (MongoDB, Express, React, Node.js). Está diseñado para gestionar equipos, tareas y proyectos de manera eficiente, priorizando la seguridad y escalabilidad.

## 🔑 Características Principales

-   **Autenticación robusta**: Sistema de registro, inicio de sesión, confirmación de cuentas por correo y recuperación de contraseña.
-   **REST API avanzada**: Endpoints para la gestión de proyectos, tareas y usuarios.
-   **Interfaces dinámicas**: Dashboards interactivos, funcionalidades de drag-and-drop, etc.
-   **Optimización**: Integración de React Query para caching.

## 💻 Tecnologías Utilizadas

-   **Frontend**: React con TypeScript, Tailwind CSS para estilos, React Router para navegación y React Query para manejo de datos
-   **Backend**: Node.js y Express, utilizando MongoDB (Mongoose) para la base de datos, JWT para autenticación, Bcrypt para hashing de contraseñas y Nodemailer para envío de correos.
-   **Otros**: Docker para contenerización, GitHub Actions para CI/CD, Swagger para documentación de la API y Jest para testing.

## 🛠️ Instalación

1. Clona el repositorio: `git clone [URL]`.
2. Configura las variables de entorno en el archivo `.env`.
3. Ejecuta `docker-compose up` para levantar MongoDB y los servicios.
4. Accede al cliente en el puerto 3000 y al servidor en el puerto 5000.

## 🚀 Despliegue

El proyecto está diseñado para ser desplegado fácilmente en plataformas como **Vercel** (frontend) y **Render** o **Heroku** (backend). Incluye un archivo `docker-compose.yml` para facilitar la configuración local.
